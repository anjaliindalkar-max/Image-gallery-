# 📸 Image Gallery Project

## 🌟 Introduction

The Image Gallery project is a modern and interactive web application developed using **HTML, CSS, and JavaScript**. It is designed to display images in a clean, organized, and responsive grid layout, making it visually appealing and easy to use. The gallery contains different categories such as animals, birds, nature, and city, allowing users to explore images in a structured way. This project focuses on combining simplicity with attractive design to create a smooth user experience.

---

## 🖼️ Project Overview

This project displays multiple images in a structured grid format where each image belongs to a specific category. The main goal of the project is to provide an easy way for users to view and explore images without confusion. The layout automatically adjusts according to screen size, making it suitable for desktops, tablets, and mobile devices. The design is kept minimal yet modern so that the images remain the main focus.

---

## 🎯 Filtering System

One of the main features of this project is the filtering system. The gallery includes buttons such as All, Animals, Birds, Nature, and City. When a user clicks on any of these buttons, JavaScript dynamically filters the images and displays only the selected category. For example, clicking on “Birds” will hide all other images except bird-related ones. This makes browsing faster, easier, and more interactive for users.

---

## 🔍 Lightbox Feature

The project also includes a lightbox feature that enhances the viewing experience. When a user clicks on any image, it opens in a large popup view with a dark background. This helps the user focus completely on the selected image without distractions. The lightbox can be closed easily by clicking the close button or by clicking outside the image area, making it user-friendly and intuitive.

---

## 🎨 Design and Styling

The design of this project is created using CSS with a focus on modern UI principles. A dark theme is used to give a professional and stylish look, while the grid layout ensures proper alignment of images. Smooth hover effects are added so that images slightly zoom in when hovered over, making the interface more interactive. Buttons are styled in a simple and attractive way to improve usability and visual appeal.

---

## ⚙️ Working Mechanism

The working of this project is based on the combination of three core technologies. HTML is used to structure the content and place images in the gallery. CSS is responsible for styling, layout, and responsiveness. JavaScript adds interactivity by controlling the filtering system and lightbox functionality. Together, these technologies create a fully functional and dynamic image gallery.
